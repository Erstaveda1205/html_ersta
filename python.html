Variabel dalam python

Variabel adalah suatu tempat yang memiliki nama dan nilai didalamnya

Nilai dari suatu variabel tertunya

  1. memiliki tipe data
  2. tersimpan dalam suatu memori di komputer
  3. dapat dioperasikan dengan operator matematika

penulisan variabel dalam python

  1. karakter pertama harus berupa huruf atau garis bawah/underscore_
  2. karakter selanjutnya dapat berupa huruf, garis bawah, atau angka
  3. Variabel tidak bolehh terdapat karakter spasi
  4. karakter pada nama variabel bersifat sensitif (case sensitif) artinya huruf kecil dan huruf besar dibedakan sebagai contoh variabel namaDepan dan namadepan adalah variabel yang berbeda

OPERATOR PYTHON

operator adalah suatu symbol yang digunakan untuk melakukan suatu operasi

sebagai contoh : 3 + 2 = 5 Dimana: 3 dan 2 bertindak sebagai operan dan + bertindak sebagai operator matematika pada operasi penjumlahan

Operator dalam python antara lain,
  1. operator aritmatika (arithmetic operators)
  2. operator perbandingan (comparison relational operators
  3. operator penugasan (assignment operators)
  4. operator logika (logical operators)

operator

Contoh

Penjelasan

Sama dengan ==

1 == 1 bernilai true

Jika masing-masing operan memiliki nilai yang sama, maka kondisi bernilai benar/true

tidak sama dengan !=

2 != 2

akan menghasilkan nilai kebalikan dari kondisi sebenarnya

tidak sama dengan

2 <> 2 bernilai false

kan menghasilkan nilai dari kebalikan dari kondisi sebenarnya

Lebih besar dari >

5 > 3 berarti True

jika nilai operan kiri lebih besar dari nilai operan kanan, maka kondisi menjadi benar

lebih kecil dari <

5 < 3 bernilai false

jika nilai operan kiri lebih besar dari nilai operan kanan, maka kondisi menjadi salah

lebih besar sama dengan >=

5 >= 3 bernilai true

jika nilai operan kiri lebih besar dari nilai operan kanan, atau sama maka kondisi menjadi benar.

lebih kecil atau sama dengan <=

5 <= 3 bernilai false

jika nilai operan kiri lebih kecil dari nilai operan kanan, atau sama maka kondisi menjadi benar

Algoritma Runtutan

Digunakan untuk program yang pernyataanya sequential atau urutan

  1. algoritma percabangan FI

    2. Digunakan untuk melakukan pemilihan atau penyeleksian kondisi sehingga tindakan diambil sesuai dengan kondisi

  2. beberapa keyword percabangan

    3. keyword percabangan diantaranya adalah if,else,elif. kondisi is digunakan untuk mengeksekusi kode jika kondisi bernilai benar
